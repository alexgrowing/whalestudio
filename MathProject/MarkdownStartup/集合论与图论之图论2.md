## 图的连通性
### 基本概念
通路(walk)：顶点与边的交替序列
$$\Gamma=v_{i_0}e_{j_1}v_{i_1}e_{j_2}v_{i_2}...v_{i_{l-1}}e_{j_l}v_{i_l}$$
其中
$$e_{j_r}=(v_{i_{r-1}},v_{i_r}),r=1,2,...,l$$
+ $v和e$是可以重复的
+ $v_{i_0}是起点,v_{i_l}是终点$
+ $通路长度|\Gamma|=l$
+ $若v_{i_0}=v_{i_l},该通路称为回路$
+ 没有重复边的通路称为简单通路
+ 没有重复边的回路称为简单回路
+ 有重复边的通路称为复杂通路
+ 有重复边的回路称为复杂回路
+ 没有重复顶点的通路称为初级通路，也叫路径
+ 没有重复顶点的回路称为初级回路，也叫圈
+ $G是含圈的无向简单图,G的周长c(G)=最长圈的长度$
  + $c\pod{K_n}=n\pod{n\ge{3}}$
  + $c\pod{K_{n,n}}=2n$
+ $G是含圈的无向简单图,G的围长g(G)=最短圈的长度$
  + $g\pod{K_n}=3\pod{n\ge{3}}$
  + $g\pod{K_{n,n}}=4\pod{n\ge{2}}$

### 定理
+ $在n阶(有向或无向)图G中,若从不同的顶点v_i到v_j有通路,则从v_i到v_j有长度小于等于n-1的通路$
+ $在n阶图G中,若从不同的顶点v_i到自身有回路,则从v_i到自身有长度小于等于n的回路$

#### 极大路径
在无向简单图中，路径的两个端点不与路径本身以外的顶点相邻，这样的路径称为极大路径
在有向图中，路径起点的前驱，终点的后继，都在路径本身上
